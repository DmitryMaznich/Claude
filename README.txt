========================================
  Smart Wash - Файлы для публикации
========================================

Содержимое папки:
-----------------
- index.html          - Главная страница сайта
- galjevica.png       - Фоновое изображение (2.1 MB)
- washing.png         - Иконка стирки (144 KB)
- drying.png          - Иконка сушки (155 KB)
- disinfection.png    - Иконка дезинфекции (237 KB)
- logo (1).svg        - Логотип сайта (14 KB)

Как загрузить на сервер:
------------------------
1. Загрузите все файлы из этой папки на ваш веб-сервер
2. Убедитесь что index.html находится в корневой директории сайта
3. Все изображения должны быть в той же папке что и index.html

Требования к серверу:
--------------------
- Любой веб-сервер (Apache, Nginx, IIS и т.д.)
- Поддержка HTML5
- Не требуется PHP, базы данных или серверной логики

Внешние зависимости:
-------------------
- Google Fonts (Cabinet Grotesk) - загружается автоматически
- Флаги стран с uxwing.com - загружаются автоматически
- Google Maps - для карты на странице

Примечания:
-----------
- Сайт полностью работает без интернета, кроме:
  * Шрифтов Google Fonts
  * Флагов стран
  * Карты Google Maps
- Все стили и скрипты встроены в index.html
- Адаптивный дизайн для мобильных устройств
