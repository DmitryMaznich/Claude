========================================
  Smart Wash - Комплект для тестирования
========================================

Содержимое папки:
-----------------
- index.html          - Главный файл сайта
- galjevica.png       - Фоновое изображение для карточки
- washing.png         - Иконка стирки
- drying.png          - Иконка сушки
- disinfection.png    - Иконка дезинфекции
- logo (1).svg        - Логотип сайта

Как запустить:
--------------
1. Скопируйте всю папку SmartWash_Package на другой компьютер
2. Откройте файл index.html в любом браузере:
   - Chrome
   - Firefox
   - Edge
   - Safari

Примечания:
-----------
- Все файлы должны находиться в одной папке
- Не требуется интернет-соединение для работы (кроме карт Google Maps)
- Сайт использует внешние шрифты Cabinet Grotesk через Google Fonts
- Флаги стран загружаются с внешнего сайта uxwing.com

Системные требования:
--------------------
- Любой современный браузер
- Рекомендуется разрешение экрана не менее 1024x768
